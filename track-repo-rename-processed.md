1. IBR-TetFund-Review-System
   This is a review system to review the proposals submitted for the annual TetFund grant for researchers.
   Frontend (Next.js, Typescript, Tailwind CSS), Backend (Node.js, Express, Typescript).
   This is deployed on a vps and the production branch is main.
   Link to repo 1: https://github.com/drid-uniben/client-new-site, old name: client-new-site, new name: ibr-tetfund-client.
   Link to repo 2: https://github.com/drid-uniben/server2_v2, old name: server2_v2, new name: ibr-tetfund-server-v2.

2. DRID-Official
   This is the official website of the Directorate of Research, Innovation and Development. It also has a section for published articles and info docs to be displayed.
   Frontend (Next.js, Javascript, Tailwind CSS), Backend (Node.js, Express, Javascript).
   This is deployed on uniben servers in their server room, we compile everything in docker and send to the Devops engineer, the production branch is main.
   Link to repo 1: https://github.com/drid-uniben/blog-frontend, old name: blog-frontend, new name: drid-official-client.
   Link to repo 2: https://github.com/drid-uniben/server, old name: server, new name: drid-official-server.
