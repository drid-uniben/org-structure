# Project Directory

## 1. IBR-TetFund-Review-System
**Description:** This is a review system to review the proposals submitted for the annual TetFund grant for researchers.
IBR stands for institution based research.
- **Stack:**
  - Frontend: Next.js, Typescript, Tailwind CSS
  - Backend: Node.js, Express, Typescript
- **Deployment:** VPS (Production branch: `main`)

**Repositories:**
- **Repo 1:** [Link](https://github.com/drid-uniben/client-new-site)
  - Old Name: `client-new-site`
  - New Name: `ibr-tetfund-client`

  - Old Name: `server2_v2`
